# Markdown essentials

| Element | Markdown Syntax |
|--|--|
| Heading | ```         \
|         | # H1        \
|         | ## H2       \
|         | ### H3      \
|         | ```         |
| Bold    | ```         \
|         | **bold text**  \
|         | ```         |
| Italic    | ```         \
|         | *italic text*  \
|         | ```         |
| Blockquote    | ```         \
|         | > blockquote  \
|         | ```         |
| Ordered list    | ```         \
|         | 1. First item \
|         | 2. Second item \
|         | 3. Third item \
|         | ```         |
| Unordered list    | ```         \
|         | - First item \
|         | - Second item \
|         | - Third item \
|         | ```         |
| Code    | ````         \
|         | ``` \
|         | Code \
|         | ``` \
|         | ````         |
| Horizontal rule    | ```         \
|         | -- \
|         | ```         |
| Link | ```         \
|         | [title](https://www.example.com) \
|         | ```         |
| Image | ```         \
|         | ![alt text](image.jpg) \
|         | ```         |
